# My_undergraduate_thesis
The implementation of the code section in my undergraduate thesis
## The code repository does not contain the data used by the code, as we do not have the authority to release these data.
## Data List
1. CRUvTS4.03 precipitation and temperature data from 1979 to 2019 (https://cds.climate.copernicus.eu/datasets/insitu-gridded-observations-global-and-regional?tab=download)
2. 16 speleothem oxygen isotope records from 14 caves, including those from this study (refer to the paper description, and you can retrieve and obtain them on the NOAA website: https://www.ncei.noaa.gov/access/paleo-search/?dataTypeId=11)
3. TraCE 21ka paleoclimate model simulation data (https://www.earthsystemgrid.org/dataset/ucar.cgd.ccsm.trace.atm.proc.monthly_ave.html)
